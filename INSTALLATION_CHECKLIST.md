## 🆘 Если что-то не работает

### Ошибка: "файл database.sql не найден"
1. [ ] Убедитесь, что вы в правильной папке
2. [ ] Выполните: `dir database.sql` (Windows) или `ls database.sql` (Mac/Linux)
3. [ ] Если файла нет, скачайте заново

### Ошибка: "Python не найден"
1. [ ] Выполните: `python --version`
2. [ ] Если не работает, установите Python с https://www.python.org/
3. [ ] Выберите: **Add Python to PATH** при установке
4. [ ] Перезагрузитесь

### Ошибка: "файл уже существует"
1. [ ] При запуске скрипта, выберите: **y** (да, переписать)
2. [ ] Или удалите старый файл: `rm transnafta.db`

### БД не открывается в DBBrowser
1. [ ] Проверьте, что файл `transnafta.db` создан
2. [ ] Размер должен быть > 50 KB
3. [ ] Попробуйте создать БД заново

### Нет данных в таблицах
1. [ ] Проверьте таблицу **services** - там должно быть 5 услуг
2. [ ] Если пусто, БД не инициализирована
3. [ ] Создайте её заново: `python init_database.py create`
