# 📖 НАВИГАЦИЯ ПО ДОКУМЕНТАЦИИ БАЗЫ ДАННЫХ

## 🎯 Где начать?

Выберите свой уровень опыта:

### 👶 Совсем новичок в базах данных?
1. Прочитайте **DB_STRUCTURE.md** (5 минут) — общее понимание
2. Следуйте **DB_SETUP_GUIDE.md** → раздел "Быстрая установка" (5 минут)
3. Откройте БД в DBBrowser и попробуйте кликать

### 👨‍💻 Опыт есть, хочу быстро начать?
1. Запустите: `python init_database.py create`
2. Откройте `transnafta.db` в DBBrowser
3. Смотрите примеры в **SQL_QUERIES_AND_API.md**

### 🚀 Опытный разработчик?
1. Прочитайте **database.sql** (структура)
2. Используйте **db.js** или **Python** для интеграции
3. Развивайте API на основе **SQL_QUERIES_AND_API.md**

## 🔑 Ключевые концепции

### Таблицы
Хранят данные (пользователи, заказы, платежи и т.д.)

### Views (Представления)
Готовые "виды" для удобного анализа данных

### Индексы
Ускоряют поиск по часто используемым полям

### Триггеры
Автоматизируют операции (обновление дат, счётчики)

### SQL запросы
Язык для получения и изменения данных

### DBBrowser
Графический интерфейс для работы с БД

### db.js
Node.js модуль с готовыми методами

## 🚀 Примеры команд

### Создать БД
```bash
python init_database.py create
```

### Показать информацию
```bash
python init_database.py info
```

### Открыть DBBrowser
```bash
# Windows
"C:\Program Files\DB Browser for SQLite\DB Browser for SQLite.exe"

# macOS
open "/Applications/DB Browser for SQLite.app"

# Linux
dbbrowser
```

### Открыть БД из командной строки
```bash
sqlite3 transnafta.db
```

### Выполнить SQL запрос
```bash
sqlite3 transnafta.db "SELECT * FROM users LIMIT 5;"
```
