## 🚀 Быстрый старт

1. **Откройте главную страницу**
   ```
   index.html
   ```

2. **Навигация**
   - Меню в верхней части
   - Ссылки на все страницы
   - Иконка 🛡️ ведёт на админ панель

3. **Пример использования калькулятора**
   - Выберите города
   - Укажите вес и объём
   - Выберите тип груза
   - Нажмите "Рассчитать и добавить в корзину"

4. **Админ панель**
   - Нажмите иконку щита в навигации
   - Выберите раздел в боковом меню
   - Просмотрите данные

## 🔐 Аутентификация и личный кабинет 🆕

### Система аутентификации
- **Регистрация**: email, пароль, ФИО, телефон
- **Вход**: email и пароль с валидацией
- **Сессии**: отслеживание входов пользователей
- **Профиль**: редактирование контактной информации
- **Безопасность**: простое хеширование паролей (можно улучшить на сервере)

### Личный кабинет (cabinet.html)
1. **Если не залогинены**:
   - Форма входа
   - Форма регистрации с валидацией

2. **После входа**:
   - Профиль пользователя с аватаром
   - История заказов с деталями (маршрут, стоимость, статус)
   - Редактирование профиля (контакты, реквизиты)
   - Управление безопасностью
   - Кнопка выхода

### Отслеживание данных 🆕
- **Сессии пользователей**: автоматическое отслеживание входов, IP, устройств
- **Действия пользователей**: переходы по страницам, клики, заполнение форм
- **Заказы**: создание, отправка, отмена, завершение
- **Логи**: полная история всех операций

Все данные сохраняются в `localStorage` и доступны в админ-панели.
