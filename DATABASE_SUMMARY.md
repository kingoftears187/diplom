## 🗄️ СТРУКТУРА БАЗЫ ДАННЫХ

### 15 ОСНОВНЫХ ТАБЛИЦ

#### Управление пользователями
- **users** — пользователи, клиенты, водители, администраторы

#### Логистика и доставка
- **orders** — заказы доставки (основная таблица)
- **services** — услуги с расценками
- **drivers** — водители с лицензиями
- **vehicles** — автомобили и техника
- **delivery_regions** — регионы обслуживания
- **order_tracking** — GPS отслеживание грузов
- **order_statuses** — справочник статусов

#### Финансы
- **payments** — платежи по заказам
- **promo_codes** — промокоды и скидки
- **promo_usage** — история использования промокодов

#### Взаимодействие с клиентами
- **reviews** — отзывы и рейтинги
- **support_tickets** — тикеты поддержки
- **ticket_messages** — переписка в тикетах

#### Аналитика
- **analytics** — ежедневная статистика
- **activity_logs** — логи всех действий

### 4 ПРЕДСТАВЛЕНИЯ (VIEWS)

- **order_summary** — сводка по заказам
- **driver_stats** — статистика водителей
- **monthly_stats** — ежемесячная статистика
- **pending_support_tickets** — открытые тикеты

### 17 ИНДЕКСОВ

Для быстрого поиска по:
- Email, статусу, дате
- ID пользователей, заказов, платежей
- Кодам и типам данных

### 5 ТРИГГЕРОВ

Для автоматизации:
- Обновления времени изменения
- Увеличения счётчиков
- Проверки ограничений

## 💻 ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ

### Node.js (db.js)
```javascript
const db = require('./db');
await db.initialize();

// Создать пользователя
const userId = await db.createUser({
    email: 'ivan@example.ru',
    password: 'password123',
    fullName: 'Иван Иванов'
});

// Создать заказ
const order = await db.createOrder({
    userId: userId,
    serviceId: 1,
    cargoDescription: 'Запчасти',
    pickupLocation: 'Москва',
    deliveryLocation: 'СПб'
});

// Получить статистику
const stats = await db.getDailyStats();
```

### SQL в DBBrowser
```sql
-- Все активные заказы
SELECT * FROM order_summary WHERE status = 'confirmed';

-- Доход за месяц
SELECT SUM(total_cost) FROM orders 
WHERE strftime('%Y-%m', created_at) = '2026-05';

-- Топ водителей
SELECT * FROM driver_stats ORDER BY completed_orders DESC;
```

### Python
```python
import sqlite3
conn = sqlite3.connect('transnafta.db')
cursor = conn.cursor()
cursor.execute('SELECT * FROM users WHERE email = ?', ('client@example.ru',))
user = cursor.fetchone()
```
