# Админ Панель ТрансНафта

## 📋 Описание

Полнофункциональная админ панель для управления системой логистической компании ТрансНафта. Доступна по адресу: `/admin.html`

## 🔐 Доступ

- **URL**: `admin.html`
- **Ссылка**: Иконка щита (⚔) в верхней навигации на всех страницах сайта
- **Текущая аутентификация**: На тестовом этапе доступна для всех

## 📊 Основные разделы

### 1. **Статистика (Dashboard)**
Главная страница с ключевыми показателями:
- ✅ Активные пользователи
- ✅ Всего заказов
- ✅ Сессий за день
- ✅ Доход за месяц
- ✅ Графики распределения заказов по типам
- ✅ Статус заказов (завершено, в пути, на рассмотрении, отменено)

### 2. **Сессии входа**
Таблица всех входов в систему:
- 👤 Имя пользователя
- 📧 Email
- 🌐 IP-адрес
- 🖥️ Устройство и браузер
- ⏰ Время входа
- 🟢 Статус сессии (активна/завершена)

**Данные для теста:**

### 3. **Пользователи**
Управление аккаунтами в системе:
- 👥 Карточки пользователей с информацией
- 📌 Роль (admin, manager, operator)
- 🟢 Статус (активен/неактивен)
- 📅 Дата присоединения
- ✏️ Редактирование
- 🗑️ Удаление
- ➕ Добавление нового пользователя

**Роли:**
- **Admin** - полный доступ
- **Manager** - управление заказами
- **Operator** - операционная поддержка

### 4. **Заказы**
Полный список всех заказов:
- 🆔 ID заказа (ТН-XXXXX)
- 👤 Клиент
- 🛣️ Маршрут (откуда → куда)
- 💰 Сумма в рублях
- 📊 Статус
  - ✅ Завершен (зелёный)
  - 🚚 В пути (синий)
  - ⏳ На рассмотрении (оранжевый)
  - ❌ Отменено (красный)
- 📅 Дата заказа

### 5. **Логи действий**
История всех операций в системе:
- 📝 Действие (вход, создание, редактирование, завершение, экспорт)
- 👤 Пользователь, выполнивший действие
- ⏰ Время операции
- 🎨 Иконки для каждого типа действия

**Типы действий:**
- 🔓 Login (вход)
- ➕ Create (создание)
- ✏️ Edit (редактирование)
- ✅ Complete (завершение)
- 📥 Export (экспорт)

### 6. **Настройки**
Конфигурация системы:

#### Система
- Название сайта
- Email поддержки
- Телефон

#### Безопасность
- Двухфакторная аутентификация (переключатель)
- Логирование действий (переключатель)
- Уведомления о входе (переключатель)

## 💾 Хранилище данных

Все данные хранятся в `localStorage` браузера для демонстрации:
- `admin_sessions` - сессии входа
- `admin_users` - пользователи
- `admin_orders` - заказы
- `admin_logs` - логи действий

## ⚙️ Функциональность

### Интерактивные элементы
✅ Переключение между вкладками  
✅ Загрузка данных при смене раздела  
✅ Статус-индикаторы (активен/неактивен)  
✅ Кнопки действий (редактировать, удалить)  
✅ Сохранение настроек  
✅ Выход из системы  

### JavaScript функции
- `initializeData()` - инициализация тестовых данных
- `switchTab(tabName)` - переключение между вкладками
- `loadSessions()` - загрузка сессий
- `loadUsers()` - загрузка пользователей
- `loadOrders()` - загрузка заказов
- `loadLogs()` - загрузка логов
- `updateDashboard()` - обновление главной панели
- `addNewUser()` - добавление пользователя
- `saveSettings()` - сохранение настроек
- `logout()` - выход

## 📱 Адаптивность

- ✅ Полная поддержка мобильных устройств
- ✅ Боковая панель скрывается на малых экранах
- ✅ Таблицы адаптивны
- ✅ Сетка карточек пользователей переходит на 1-2 колонки

## 🔄 Пример использования

1. Откройте `admin.html` или нажмите на иконку щита в навигации
2. Выберите нужный раздел в боковом меню
3. Просмотрите данные в таблицах/карточках
4. Используйте кнопки действий для управления
5. Нажмите "Выход" для завершения сессии
