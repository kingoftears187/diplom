### 🔧 УТИЛИТЫ И МОДУЛИ (2 файла)

#### 3. **`init_database.py`** (~8 KB)
- **Язык**: Python 3.7+
- **Назначение**: Утилита для создания и управления БД
- **Возможности**:
  - Создание БД из SQL скрипта
  - Просмотр информации о БД
  - Проверка целостности
  - Интерактивное меню
- **Использование**:
  ```bash
  python init_database.py create    # Создать БД
  python init_database.py info      # Показать информацию
  python init_database.py verify    # Проверить целостность
  python init_database.py           # Интерактивное меню
  ```
- **Статус**: ✅ Готово

#### 4. **`db.js`** (~12 KB)
- **Язык**: JavaScript (Node.js)
- **Требования**: sqlite3 пакет (`npm install sqlite3`)
- **Назначение**: Node.js модуль для работы с БД
- **Методы**: 30+ готовых методов для:
  - Управления пользователями (createUser, getUserByEmail, updateUserProfile)
  - Управления заказами (createOrder, getUserOrders, updateOrderStatus)
  - Отслеживания (addTrackingPoint, getTrackingHistory)
  - Платежей (createPayment, completePayment)
  - Отзывов (createReview, getReviews)
  - Аналитики (getDailyStats, getMonthlyStats)
  - Поддержки (createTicket, addTicketMessage)
  - Промокодов (validatePromoCode, usePromoCode)
  - Логирования (logActivity)
- **Использование**:
  ```javascript
  const db = require('./db');
  await db.initialize();
  const user = await db.getUserByEmail('email@example.ru');
  ```
- **Статус**: ✅ Готово
