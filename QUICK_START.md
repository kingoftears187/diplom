# 🚀 Быстрый старт для ТрансНафта

## 📖 Для пользователей

### Первый вход
1. Откройте **index.html**
2. Кликните "Кабинет" в меню
3. Заполните форму регистрации:
   ```
   ФИО:     Иван Петров
   Email:   ivan@example.com
   Пароль:  MyPassword123
   Телефон: +7 (999) 123-45-67
   ```
4. Нажмите "Зарегистрироваться"
5. Введите ваши данные для входа
6. Готово! 🎉

### Создание заказа
1. Нажмите "Калькулятор" в меню
2. Выберите города отправления/назначения
3. Укажите вес и объём груза
4. Выберите тип груза
5. Нажмите "Рассчитать и добавить в корзину"
6. Перейдите в "Корзину"
7. Нажмите "Оформить заявку"
8. Заказ появится в вашем кабинете! ✅

### Просмотр профиля
1. Откройте **cabinet.html**
2. Войдите в систему
3. Нажмите "Редактировать"
4. Измените свои данные:
   - Телефон
   - Компания
   - Адрес
   - Реквизиты (ИНН, КПП, БИК)
5. Нажмите "Сохранить"

## 👨‍💼 Для администратора

### Просмотр статистики
1. Откройте **index.html**
2. Кликните на щит 🛡️ в меню
3. На странице admin.html:
   - **Активные пользователи**: количество пользователей в сети
   - **Всего заказов**: общее количество заказов
   - **Сессии сегодня**: входов за сегодня
   - **Ежемесячный доход**: сумма всех заказов

### Просмотр сессий входа
1. Откройте **admin.html**
2. Перейдите на вкладку "Сессии"
3. Смотрите таблицу с:
   - ID сессии
   - Email пользователя
   - IP адрес
   - Устройство
   - Время входа

### Просмотр заказов
1. Откройте **admin.html**
2. Перейдите на вкладку "Заказы"
3. Смотрите таблицу с:
   - ID заказа (ТН-XXXXXX)
   - Маршрут (город → город)
   - Стоимость
   - Статус (На рассмотрении, Отправлен, В пути, Завершен)
   - Дата создания

### Просмотр логов действий
1. Откройте **admin.html**
2. Перейдите на вкладку "Логи"
3. Смотрите последние 20 действий:
   - Загрузки страниц
   - Клики по кнопкам
   - Заполнение форм
   - Отправка заказов
   - Входы/выходы

## 🔑 API (для разработчиков)

### Аутентификация

```javascript
// Регистрация
window.auth.register(email, password, fullName, phone)
// Возвращает: { success, message, user }

// Вход
window.auth.login(email, password)
// Возвращает: { success, message, user }

// Выход
window.auth.logout()
// Возвращает: { success, message }

// Получить текущего пользователя
window.auth.getCurrentUser()
// Возвращает: { id, email, fullName, phone, avatar, ... }

// Проверить, залогинен ли
window.auth.isLoggedIn()
// Возвращает: true/false

// Обновить профиль
window.auth.updateProfile({ phone, profile: {...} })
// Возвращает: { success, message }
```

### Отслеживание

```javascript
// Начать сессию (автоматически при загрузке)
window.sessionTracker.startSession()

// Записать действие
window.sessionTracker.recordAction('action_name', { extra_data })

// Получить текущую сессию
window.sessionTracker.getCurrentSession()

// Завершить сессию
window.sessionTracker.endSession()

// Создать заказ
window.orderTracker.createOrder({
    from: 'Москва',
    to: 'СПб',
    weight: '500',
    volume: '2',
    cargoType: 'Общий',
    price: '125000'
})

// Отправить заказ
window.orderTracker.submitOrder(orderId)

// Отменить заказ
window.orderTracker.cancelOrder(orderId)
```

## ⚠️ Важные замечания

1. **Данные в браузере**: Все данные хранятся в localStorage браузера
2. **Очистка**: При очистке истории браузера данные удалятся
3. **Разные браузеры**: Данные не синхронизируются между браузерами
4. **Безопасность**: Это только тестовое приложение, не используйте реальные пароли
5. **Хеширование**: Пароли хешируются локально, это не криптографически безопасно

## 🆘 Часто задаваемые вопросы

**Q: Забыл пароль, что делать?**
A: Создайте новый аккаунт с другим email

**Q: Где мои данные?**
A: В localStorage браузера (F12 → Application → Local Storage)

**Q: Почему я не вижу свои заказы?**
A: Убедитесь, что вы залогинены и заказы созданы под этим аккаунтом

**Q: Как удалить аккаунт?**
A: В кабинете → Безопасность → Удалить аккаунт

**Q: Можно ли использовать это в продакшене?**
A: Нет, это только демонстрационная система. Нужен реальный сервер и БД.
